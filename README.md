# 自我工作到退休金試算工具

這是一個 Streamlit 網頁版小工具，用來估算從開始工作到退休時可能可領的退休相關現金流。

工具把三件事分開算：

- 勞保老年年金：退休後每月領的年金，依官方兩式擇優估算。
- 勞退新制個人專戶：雇主提繳與個人自提累積出的專戶餘額，再用預估領取年限平均攤提。
- 勞退舊制退休金：依勞基法第 55 條基數規則估算一次退休金，適用舊制或新舊制混合情境。

## 需要填的變數

最少建議填：開始工作年齡、目前年齡、預計退休年齡、工作第一年月薪、工作第一年勞就保投保薪資、目前月薪、目前勞就保投保薪資。

完整試算建議填：

1. 開始工作年齡
2. 目前年齡
3. 預計退休年齡
4. 預估領到幾歲
5. 工作第一年月薪資總額
6. 工作第一年勞就保月投保薪資
7. 目前月薪資總額
8. 目前勞就保月投保薪資
9. 未來薪資年成長率
10. 勞退制度：新制、舊制、新舊制混合
11. 保留舊制年資，如果是新舊制混合
12. 退休前平均工資，如果要估算舊制基數
13. 目前已累積勞退個人專戶餘額
14. 雇主勞退提繳率
15. 個人自願提繳率
16. 勞退帳戶年化收益率假設
17. 年金提前或展延請領年數
18. 最高 60 個月平均月投保薪資，如果你已經知道

## 本機執行

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Streamlit Cloud 部署

這是最適合目前版本的公開上線方式，部署後會得到一個公開網址，可以直接分享給別人使用。

1. 到 GitHub 建立一個新的 repository。
2. 將本資料夾內的檔案上傳到該 repository。
3. 到 Streamlit Community Cloud 建立新 App。
4. 選擇剛剛的 GitHub repository。
5. Main file path 填 `streamlit_app.py`。
6. Deploy 後即可取得公開網址。

需要上傳的檔案：

- `streamlit_app.py`
- `retirement_calculator.py`
- `requirements.txt`
- `README.md`
- `.streamlit/config.toml`

不需要上傳的檔案：

- `.venv/`
- `.pytest_cache/`
- `__pycache__/`

目前工具沒有帳號登入，也沒有資料庫，使用者在網頁上輸入的資料只會用於當次瀏覽器試算，不會儲存在伺服器資料庫。

## 資料與限制

- 勞保投保薪資分級表採民國 115 年 1 月 1 日起適用級距。
- 勞保老年年金公式採官方 A/B 兩式擇優。
- 勞退舊制採勞基法第 55 條基數規則：前 15 年每年 2 個基數，超過 15 年每年 1 個基數，最高 45 個基數。
- 勞退新制月提繳工資表在工具中採簡化估算：用月薪作為提繳工資基礎，並以 29,500 到 150,000 控制上下限；若要精算，後續可補完整月提繳工資分級表。
- 新舊制混合情境下，舊制年資是否成立取決於實際雇主、保留年資與個案紀錄，本工具只能做情境估算。
- 本工具只做情境試算，實際金額以主管機關核定資料為準。

## 主要來源

- 勞工保險局：勞工保險投保薪資分級表，115 年 1 月 1 日起適用。
- 勞工保險局：老年年金給付標準。
- 勞工保險局：勞工退休金月提繳分級表，115 年 1 月 1 日生效。
- 勞動部勞動法令查詢系統：勞動基準法第 55 條。
